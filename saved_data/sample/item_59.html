          1. GitHub user dguy opened a pull request: https://github.com/apache/kafka/pull/2244 KAFKA-4490: Add Global Table support to Kafka Streams Add Global Tables to KafkaStreams. Global Tables are fully replicated once-per instance of KafkaStreams. A single thread is used to update them. They can be used to join with KStreams, KTables, and other GlobalKTables. When participating in a join a GlobalKTable is only ever used to perform a lookup, i.e., it will never cause data to be forwarded to downstream processor nodes.
